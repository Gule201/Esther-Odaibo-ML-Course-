# Assignment 8: Natural Language Processing in Healthcare

## Problem
This assignment focused on critiquing how NLP is used in healthcare to process unstructured clinical text.

## Approach
I explained how NLP extracts information from clinical notes, reviewed its benefits, and discussed limitations such as accuracy, privacy, and bias.

## Results
The assignment showed that NLP can improve efficiency and clinical insight by turning narrative data into structured information.

## What I Learned
I learned that NLP has strong potential in healthcare, but it must be implemented carefully to avoid errors and inequitable outcomes.
