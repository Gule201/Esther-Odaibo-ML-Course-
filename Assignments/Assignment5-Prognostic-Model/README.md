# Assignment 5: Critiquing a Prognostic Model

## Problem
This assignment focused on evaluating a machine learning model designed to predict sepsis in ICU patients.

## Approach
I reviewed the variables used in the model, assessed its strengths and limitations, and compared it with traditional prediction methods.

## Results
The model showed strong predictive performance and demonstrated the value of using dynamic clinical data for early sepsis detection.

## What I Learned
I learned that predictive models can improve care, but they must be validated carefully and integrated thoughtfully into clinical workflows.
