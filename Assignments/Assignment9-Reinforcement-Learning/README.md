# Assignment 9: Reinforcement Learning in Healthcare

## Problem
This assignment explored how reinforcement learning could be applied to vasopressor dosing in ICU patients with septic shock.

## Approach
I described the RL scenario using states, actions, and rewards, and explained how this approach could support real-time treatment decisions.

## Results
The assignment showed that reinforcement learning could help personalize ICU treatment strategies and improve outcomes in complex situations.

## What I Learned
I learned that reinforcement learning offers exciting healthcare applications, but safety, validation, and human oversight are essential.
